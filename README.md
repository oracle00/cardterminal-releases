# CardTerminal

Demul エミュレータ（Hikaru/Virtual On 4）向けのカードデータ監視・書き換えツールです。  
ソースコードは非公開。
自分が使う目的で作成しているので使用は自己責任で。

## 動作環境

- Windows 10 / 11 (64 bit)
- .NET Flamework 8.0 必須
- Demul エミュレータ 2025年12月20日版

## 機能

| 機能 | 説明 |
|------|------|
| Demul 起動 & 接続 | 指定コマンドで Demul を起動しプロセスハンドルを取得 |
| カードデータ監視 | 1 秒ごとにメモリを読み取り、変化があれば画面に表示（点滅通知） |
| メタデータ表示 | PlayerName / NickNameCode / Rank / Unit / ColorID / Date をデコードして表示 |
| ゲーム内のカードデータ更新 | 選択したファイルのカードデータをメモリへ書き込み |
| ゲーム内のカードデータ保存 | 現在のカードデータをメタデータ付きで保存 |
| ファイルエクスプローラー | ドライブ選択対応の内蔵エクスプローラーでカードファイルを閲覧・選択 |
| UnitID 変更 | プルダウンでUnitIDを変更、Lock Unit で固定。ゲーム内のカード排出時に有効化 |
| ColorID 変更 | プルダウンでカラーIDを変更、Lock Color で固定。ゲーム内のカード排出時に有効化 |
| Card Value Fix | カード使用回数を定期的に復元（1 秒ごとに指定アドレスへ書き込み） |

## 使い方

### 初回起動

初回起動時に demul.exe の場所を選択するダイアログが表示されます。  
demul.exe を選択すると起動コマンドが自動設定され、次回以降は自動的に読み込まれます。

### 基本的な操作

#### DemulとVO4thの起動

**Start Demul & Connect** をクリックして Demul を起動・接続する

#### カードデータの保存方法

ゲーム内で**カードの取り出しダイアログが表示されているタイミング**で **Save card data** ボタンを押すとメタデータとカードデータをファイルに保存する

#### カードデータの読み込み方法

1. 右側のファイルエクスプローラーで読み込みたいファイルをクリック
2. **Change Card Data** をクリックするとゲーム内のカードデータが更新される
3. Demulのショートカットキーでカードを挿入する

### その他の機能

#### 機体データ上書き

推奨手順
1. VO4thをVendorModeで起動
2. タイトル画面が表示されたら機体データを変更したいカードを挿入
3. ゲーム内で**VERTUAL-ON FORCE TERMINAL**の画面が表示されるのを待つ
4. ツール上のUnitID プルダウンで機体を変更、Lock Unitにチェックを入れる
5. ゲーム内で**カード排出、終了**を選択すると指定した機体に変更される
6. 再度カードを挿入して機体が変更されていることを確認**Save card data**ボタンでカードデータをファイルに保存

#### 機体カラーデータ上書き
推奨手順
1. VO4thをVendorModeで起動
2. タイトル画面が表示されたら機体カラーを変更したいカードを挿入
3. ゲーム内で**VERTUAL-ON FORCE TERMINAL**の画面が表示されるのを待つ
4. ツール上のColorID プルダウンでカラーを変更、Lock Colorにチェックを入れる
5. ゲーム内で**カード排出、終了**を選択すると指定したカラーに変更される
6. 再度カードを挿入してカラーが変更されていることを確認**Save card data**ボタンでカードデータをファイルに保存

#### カード使用回数30固定

Enable Restore Card Uses を有効にするとカード使用回数が自動復元される
